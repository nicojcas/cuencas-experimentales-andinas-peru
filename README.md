Este repositorio contiene los resultados de la validación de una red de sensores de nivel en cuencas experimentales andinas del Perú, mediante modelos hidrológicos lluvia–escorrentía.

GR2M/

 - Caudales mensuales observados obtenidos a partir del producto ARNOVIC.

 - Caudales mensuales simulados mediante el modelo GR2M.

 - Todos los caudales están expresados en m³/s.

GR4J/

 - h: niveles de agua observados (m).

 - q_obs: caudales observados derivados del producto ARNOVIC (m³/s).

 - q_sim_arn: caudales simulados mediante una calibración tradicional del modelo GR4J (m³/s).

 - q_sim_irc: caudales simulados mediante una calibración basada en curva de descarga inversa (m³/s).
